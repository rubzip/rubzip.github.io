Fast, minimalist, and highly customizable Astro template for researchers, professors, and PhD students. Strongly inspired by Academic Pages and AstroPaper.

Key features:
- **Markdown-Driven:** All content configured via markdown frontmatter
- **Academic Features:** LaTeX support, DOI/ISBN metadata, publication tracking
- **Design:** 2-column layout, light/dark mode, responsive

Tech stack: Astro v6, TypeScript, Tailwind CSS v4

Repository: [https://github.com/rubzip/academic-portfolio-astro](https://github.com/rubzip/academic-portfolio-astro)
